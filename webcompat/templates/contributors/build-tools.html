Build Tools

The webcompat.com project is an open source project. It relies on the effort of the Mozilla webcompat team and contributions from volunteers across the world. The Microsoft Edge and Google Chrome teams also participate from time to time.

There are different areas where you can create an impact by developing tools, including:

  1. The webcompat.com website. It is a mix of Python and JavaScript, CSS, HTML using GitHub as the backend. Anyone with UX, design, coding, writing or other skills are always welcome to contribute.
  2. The webcompat reporter extensions help to make reports directly accessible in browsers.

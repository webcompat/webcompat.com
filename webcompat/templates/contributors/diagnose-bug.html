• Find a bug with a "needsdiagnosis" label or that is otherwise untriaged
  • Check to see if bug has been reproduced already
  • If not, try to reproduce the bug
  • If bug is NOT reproducible, write a comment on the issue
  • If bug is reproducible, try to reproduce on other browsers or OS
  • Check the console to see if there are errors
  • Google any errors
  • Write a comment with your findings on the issue
  • If you know how to fix the issue, or have information related to a fix, add it in a comment
  • If not, post what you tried on the issue

There are a wide list of references on how to diagnose bugs for desktop and mobile.

  • Firefox Remote Debugging (Android and Firefox OS)
